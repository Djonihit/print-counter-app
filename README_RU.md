# Print Counter App

Android приложение для подсчёта принтов и вышивки с функциями статистики и истории.

## Основные возможности

### 📊 Счётчик
- Подсчёт печатей в форматах: **A2, A3, A4, A5, A6**
- Подсчёт вышивки в форматах: **A3, A4, A5, A6**
- Быстрое добавление записей через простой интерфейс
- Увеличение/уменьшение количества с помощью кнопок

### 📈 Статистика
- **Общая статистика** - всего принтов и вышивки
- **Ежедневная статистика** - подсчёт за текущий день
- **Недельная статистика** - данные за последние 7 дней
- **Месячная статистика** - подробный анализ за месяц

### 📋 История
- Полный список всех записей
- Сортировка по времени (новые в начале)
- Информация о формате, количестве и времени
- Возможность удаления записей

### 🎨 Интерфейс
- Современный Material Design 3
- Светлая/тёмная тема
- Удобная навигация через нижнее меню
- Быстрая и отзывчивая работа

## Технический стек

- **Язык**: Kotlin
- **Архитектура**: MVVM + Clean Architecture
- **UI**: Jetpack Compose
- **База данных**: Room Database
- **Асинхронность**: Kotlin Coroutines
- **Версия Android**: 8.0+ (API 26)

## Экраны приложения

### 🏠 Счётчик (главный экран)
- Две отдельные секции: Печать и Вышивка
- Выбор формата для каждого типа работ
- Счётчик с кнопками +/-
- Кнопка добавления записи в БД

### 📊 Статистика
- Общие итоги (всего принтов и вышивки)
- Статистика за сегодня
- Недельная сводка в виде таблицы

### 📝 История
- Список всех записей в хронологическом порядке
- Отображение формата, количества и времени
- Кнопка удаления для каждой записи

## Установка и запуск

1. Клонируйте репозиторий
```bash
git clone https://github.com/Djonihit/print-counter-app.git
```

2. Откройте проект в Android Studio

3. Синхронизируйте Gradle
```bash
./gradlew sync
```

4. Запустите приложение
- Выберите эмулятор или подключённое устройство
- Нажмите Run или Shift+F10

## Требования

- **Android Studio**: Flamingo или новее
- **Android SDK**: 26 (Android 8.0) или новее
- **Kotlin**: 1.9.20
- **Gradle**: 8.1.4

## Использование

### Добавление записи о печати
1. На главном экране выберите формат (A2-A6)
2. Установите количество с помощью кнопок +/-
3. Нажмите "Добавить печать"
4. Запись сохранится в БД

### Добавление записи о вышивке
1. На главном экране выберите формат (A3-A6)
2. Установите количество
3. Нажмите "Добавить вышивку"

### Просмотр статистики
1. Перейдите на вкладку "Статистика"
2. Просмотрите общие итоги и дневную статистику
3. Скролльте вниз для просмотра недельных данных

### Просмотр истории
1. Перейдите на вкладку "История"
2. Просмотрите все записи в порядке новизны
3. Нажмите "Удалить" для удаления записи

## База данных

Данные хранятся локально на устройстве в Room Database. Каждая запись содержит:
- ID (автогенерируемый)
- Количество единиц
- Формат (A2, A3, A4, A5, A6)
- Тип работы (Печать или Вышивка)
- Описание (опционально)
- Время создания записи
- Имя устройства (опционально)
